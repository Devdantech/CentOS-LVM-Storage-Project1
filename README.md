My LVM Project

![Step 1](images/1.png)  
Create a virtual machine using VirtualBox, and apply your own configurations
![Step 3](images/4.png)  
Enter the setting parameters of your VM, next to (Controller SATA), select the hard disk icon
![Step 4](images/5.png)  
In the Hard Disk selector, locate the icon (Create New)& select it
![step 5](images/6.png)  
In a virtual hard disk, name the file, determine its size, and specify the file format. Here we are using 20Gb hard disk
![Step 6](images/8.png)  
Within storage parameters,verify that the hard disk is created. Now, you are ready to launch the VM
